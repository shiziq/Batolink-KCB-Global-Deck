## How to Present

1. Open `Batolink Global Presentation Deck/index_v4.html` in **Google Chrome** or **Microsoft Edge**
2. Press **F11** for full-screen presentation mode
3. Use **→ arrow key** or **space bar** to advance slides
4. On Slide 10, click the **play button** to start the product demo video
5. Refer to `SPEAKER_NOTES_v4.md` for scripted talking points

### Navigation
- **Arrow keys / Space:** Next slide
- **Swipe gestures:** On tablets
- **Navigation dots (right side):** Jump to any slide
- **ESC:** Does nothing (prevents accidental exit)

## Key Financial Highlights

- **Per-transaction revenue:** KSh 8.08 (0.95% on KSh 850 average)
- **Pilot daily revenue:** KSh 96,960 (12,000 transactions)
- **Break-even point:** Month 2.4
- **Year 1 projection:** KSh 2.4 billion at 47-branch scale
- **Fee advantage:** 0.95% vs M-Pesa 1.5-2.5% vs bank cards 2.0-3.5%
- **Free trial:** First 3 months at zero cost to KCB
